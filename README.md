# ProfiFinder RN — Коммит 2/3: данные и навигация

**feat: вынес данные в модуль, подключил фильтры, переходы и экран профиля**

Второй шаг. Захардкоженный UI из коммита 1 переведён на общий источник данных,
фильтры реально фильтруют, экраны связаны навигацией.

## Что добавилось
- `src/data/catalog.js` — все эксперты, категории и фильтры в одном месте.
- Вынесены компоненты `Chip.jsx` и `ExpertCard.jsx`.
- Каталог на `FlatList` с рабочими фильтрами по теме и формату + поиск по тексту.
- Экран профиля `src/app/expert/[id].jsx` (push поверх вкладок, кнопка «назад»).
- Переходы: категория/частый запрос/поиск → каталог с параметрами; карточка → профиль.
- Кнопка «Получить контакты» на профиле.

## Чего ещё нет (будет в коммите 3)
- Тёмной темы.
- Пустого состояния каталога и мелкой полировки (тени, состояния нажатия).
