# ProfiFinder RN — Коммит 1/3: каркас и статичный UI

**feat: перенёс экраны на React Native (Expo Router + NativeWind), пока статикой**

Первый шаг ручного переноса с веба. Поднят Expo-проект, настроен NativeWind,
собран визуальный каркас всех трёх вкладок. Данные и логика — на следующем шаге.

## Что уже есть
- Нижние вкладки: **Поиск · Каталог · Стать экспертом** (`src/app/(tabs)`).
- Экраны свёрстаны на NativeWind, светлая тема.
- Лого-лупа `src/components/Logo.jsx`.

## Что ещё заглушки (сделаю в коммите 2–3)
- Карточки и категории захардкожены прямо в экранах.
- Поиск и фильтры — визуальные, не фильтруют.
- Нет экрана профиля эксперта и переходов между экранами.
- Нет тёмной темы.

## Запуск
```bash
npm install
npx expo start -c      # затем нажать "a" (эмулятор) или сканировать QR в Expo Go
```
